Inventory & parts
critical: always use c_fifoix summed bin quantities for inventory. DO NOT use qoh_cache. qoh_cache it is a cached value that lags behind real data and will give you wrong numbers.
TableWhat It IsNotes
PartsMaster list of every part in the systemcore table always your starting point for part lookups
c_fifoixFIFO inventory index — live bin quantities per partuse this SUM(qty) grouped by partno = reliable QOH

Fifoix PiecesPiece/serial level detail within FIFO inventory
Part Pick LayersTracks which FIFO inventory layers were consumed by a work order pick
