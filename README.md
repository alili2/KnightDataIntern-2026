# KnightDataIntern-2026

Onboarding site for the Knight Aerospace Data Team summer internship (June–August 2026).

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Welcome, quick info, 3-month plan, week 1 SQL tasks |
| Database Guide | `database.html` | Key Cetec ERP tables, columns, known quirks |
| Metabase 101 | `metabase.html` | How to navigate, write queries, build dashboards |
| Cetec ERP | `cetec.html` | What Cetec is, how it works, known problems |

## Setup (GitHub Pages)

1. Push this repo to GitHub under your account
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://yourusername.github.io/KnightDataIntern-2026`
